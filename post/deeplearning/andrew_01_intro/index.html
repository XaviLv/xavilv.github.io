Preface

After trying to learn all the stuff about deep learning in a short time, I found myself gradually lost like inside a trap never ends. Although heard a lot about Andrew Ng, I just read some slides from him.

Fortunately, I found the video online course presented by Master Andrew, with the support from Netease and deeplearning.ai. Since a thousand miles begin with a single step, why don’t I?

What is a neural network?

  • It’s a power learning algorithm inspired by how the brain works.

Why is it taking off now?

  • As known, Deep Learning is not a fresh new technique. It bursts out in these years due to the rapic growth of massive big data technology and hardware computation capabilities. So that we can do training much more complicated networks in much shorter time.

Suppervised Learning for Neural Network

Regression v. Classification

  • Regression

    • predicting results in a continous output
    • mapping input variables to some continuous function

  • Classification:

    • predicting results in a discrete output
    • mapping input variables into discrete categories

Different Types of Neural Network

  • CNN: image application
  • RNN: one-dimensional sequence data
  • Hybrid NN: autonomous driving, .etc
  • Neural Network can performance explicitly better at its growing scale while the amount of data grows!
  • Howerver, when the data source is very limited, Traditional ways such as SVM could also be considered.

Structured vs Unstructured Data

  • Structured: data that has a defined meaning such as price
  • Unstructured: data like pixel, raw audio, text
