为什么要安装 Nvidia Docker？为啥不直接在 Linux 上安装 CUDA？有什么好处或者必要性吗？

假如我有两个项目，一个是pytorch实现需要CUDA11，一个是tensorflow实现需要CUDA9，这个时候就需要docker了。但是记住，docker里面的 cuda 版本要比外面的 cuda 版本低，如果反过来 docker 里面的 cuda 版本比外面的高就跑不起来了。

详情需要进一步了解docker

安装 CUDA

  1. 第一件事：disable nouveau，可能要上网搜和尝试各种方法直到完成。这个 nouveau 是 ubuntu 自带的开源驱动，如果不 disable 是装不了 CUDA 的；

  2. CUDA Download：一定要选择用 runfile 安装

  3. 不能直接用官网的wget命令安装，必须加上 --no-opengl-libs, 否则很可能导致系统重装.

但是有图像界面的时候不能直接用这个命令，需要先把界面drop掉，比如用 sudo init 3（不知所云，需要试试……）

1

sudo sh cuda_11.2.0_460.27.04_linux.run —no-opengl-libs
