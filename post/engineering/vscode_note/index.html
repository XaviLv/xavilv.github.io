Adding File Header

Normally if you create a new file like *.py or *.md, it’s better to have the file header added automatically. Fortunately most IDEs can do this. In VS Code it’s easy by the help of User Snippets.

1. Edit settings.json

Ctrl + P to open settings.json, then add "[markdown]":{"editor.quickSuggestions": true} in json body.

2. Open VScode / Preferences / User Snippets

3. New a snippets

4. Edit content like this, for example a markdown file.

"My File Header": {
		"scope": "markdown",
		"prefix": "fileheader",
		"body": [
			"---",
			"title: \"\"",
			"date: ${CURRENT_YEAR}-${CURRENT_MONTH}-${CURRENT_DATE}T${CURRENT_HOUR}:${CURRENT_MINUTE}:${CURRENT_SECOND}+08:00",
			"lastmod: ${CURRENT_YEAR}-${CURRENT_MONTH}-${CURRENT_DATE}T${CURRENT_HOUR}:${CURRENT_MINUTE}:${CURRENT_SECOND}+08:00",
			"draft: false",
			"tags: [\"math\"]",
			"categories: [\"Engineering\"]",
			"summary: \"An example set of mathematic annotations.\"",
			"---"
		],
		"description": "this is a description"
	}

5. Import the file header in a new file by typing fileheader.

Then you would got something like this, bravo!
